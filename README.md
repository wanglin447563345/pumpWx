# pumpWx
一个水泵监测微信小程序<br>
# # # 技术栈：
echarts小程序版<br>
weui小程序版<br>
# # # 功能：
简单的模拟了上啦数据加载<br>
自定义了弹窗添加组<br>
实现滑动编辑删除<br>
模拟了组删除<br>
